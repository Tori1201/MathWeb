Welcome to Pre-Calculus

What is Pre-Cal?

Pre-Calculus covers algebra and trigonometry at an advanced level in order to prepare you for Calculus. It covers topics such as limits, functions, and coordinate systems. This subject helps develop pattern recognition skills to use in advanced math courses.

Key Topics in Pre-Calculus

  • Functions: Learn the behaviors and different types of functions..
  • Limits: Learn about the value of a function as in approaches a specific point.
  • Solving right and oblique triangles: Use the Pythagorean theorem and trigonometric functions to find missing information..
  • Polar coordinate systems: Convert between coordinate types.
